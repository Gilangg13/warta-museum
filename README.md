## Run Locally

1. Clone the project
    ```bash
      git clone https://github.com/Gilangg13/warta-museum.git
    ```

2. Go to the project directory
    ```bash
      cd warta-museum
    ```

3. Install dependencies
    ```bash
      npm install
    ```

4. Build the project 
    ```bash
      npm run build
    ```

5. Start the server
    ```bash
      npm run start-dev
    ```
